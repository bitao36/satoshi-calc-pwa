Características

  • Para obtener el precio de las divisas se usa la API de yadio.io
  • El precio se actualiza cada minuto.
  • Si se pierde la conexión a internet, se usa el último valor obtenido.
  • La divisa por defecto se puede configurar haciendo click en el icono de configuración.

Motivación

Creé esta sencilla aplicación ya que ninguna de las calculadoras actuales cubria mis necesidades y la quise compartir con todos aquellos bitcoiners que la deseen usar.
